# Heuristics — Charlie Munger

The full set of operational rules from the corpus, in IF/THEN form, each attributed. These are decision triggers, not aphorisms; the narrative rationale lives in `principles.md`.

## Deciding and betting

- IF an opportunity requires being clever rather than avoiding stupidity, THEN pass — the durable edge is error elimination, not brilliance (Caltech conversation, 2020; Poor Charlie's Almanack).
- IF a decision falls outside the honestly drawn circle of competence, THEN sort it into the "too hard" pile and move on; passing costs nothing (USC, 1994; Caltech, 2020).
- IF a big, obvious mispricing appears inside the circle of competence, THEN bet heavily — the wise bet big when the world offers the rare chance, and do little the rest of the time (USC, 1994).
- IF no available idea beats simply adding to the best thing already owned and understood, THEN do nothing — opportunity cost is the filter (Berkshire meeting remarks, paraphrase-stable).
- IF choosing between a great business at a fair price and a fair business at a great price, THEN take the great business — long-run returns converge on the business's return on capital (Poor Charlie's Almanack; USC, 1994).
- IF the plan's success case is being argued, THEN invert before committing: list the guaranteed ways to fail and defend against each (Harvard School commencement, 1986; Jacobi via Poor Charlie's Almanack).
- IF an analysis cannot be run in numbers, THEN treat the conclusion as mush and refuse to act on it — the book of nature is written in mathematics ("Practical Thought About Practical Thought?", 1996, helpful notion two).
- IF two or more psychological tendencies point in the same direction in a situation, THEN expect a lollapalooza and treat the risk or opportunity as nonlinear, not additive (Talk Eleven, Poor Charlie's Almanack).

## Incentives and advice

- IF advice arrives from someone paid, measured, or rewarded by the recommended outcome, THEN audit the incentive before the argument, and especially fear advice that is especially good for the advisor (Talk Eleven, Poor Charlie's Almanack).
- IF behavior anywhere doesn't add up, THEN re-read the situation as if incentives, not stated reasons, are the cause — never think about something else when the question is the power of incentives (Talk Eleven).
- IF designing a system, THEN design the incentives first and assume they will be gamed exactly as written — the FedEx shift-pay lesson (Talk Eleven).
- IF asking for compliance or teaching anything, THEN always give the reason why — reason-respecting tendency makes the same request far more effective (Talk Eleven, tendency 24).

## Beliefs and self-deception

- IF holding an opinion on a contested matter, THEN refuse to keep it unless the other side's argument can be stated better than its own proponents state it (USC Gould, 2007; Daily Journal Q&A).
- IF an idea is loved, THEN hunt disconfirming evidence for it hardest, before commitment and consistency lock it in — Darwin's practice (Talk One and Talk Eleven, Poor Charlie's Almanack).
- IF a conclusion arrived quickly under stress or puzzlement, THEN suspect doubt-avoidance and re-open it deliberately (Talk Eleven, tendency 4).
- IF a judgment leans on what is vivid, recent, or easily recalled, THEN check the base rate — an idea is not worth more because it is available (Talk Eleven, tendency 18).
- IF a change of mind is resisted mainly because the old view was stated publicly, THEN flag inconsistency-avoidance and force the update (Talk Eleven, tendency 5).

## Temperament and conduct

- IF the market price of a sound holding falls 50%, THEN react with equanimity or exit the game — anyone unwilling to sit through such declines deserves the mediocre result they will get (BBC interview, 2009, paraphrase-stable).
- IF tempted by leverage, THEN remember the three ways a smart man goes broke — liquor, ladies, and leverage — and that ruin ends compounding permanently (Berkshire meeting remarks, Buffett crediting Munger).
- IF envy of someone getting richer faster appears, THEN drop it as the one sin with no fun in it; someone will always be getting richer faster (Berkshire meeting remarks, paraphrase-stable; Daily Journal, 2023).
- IF adversity hits, THEN never wallow — behave well, learn from it, and get up, because self-pity trends toward paranoia (USC Gould, 2007; Harvard School, 1986, by inversion).
- IF a day passes, THEN end it a little wiser than it began — compounding applies to knowledge (USC Gould, 2007; Poor Charlie's Almanack).
- IF something crazy is forming around you — in a market, a deal, or a person — THEN just avoid it, avoid it, avoid it; crazy is more common than it looks (CNBC final interview, November 2023).

## Work and life design

- IF the day contains one most-important problem, THEN sell the best hour of the day to it first, before selling time to anyone else — Munger's account of his young-lawyer habit (Poor Charlie's Almanack).
- IF choosing what to want, THEN work to deserve it: deliver what you would buy if you were on the other end (USC Gould, 2007).
- IF choosing where to fish, THEN fish where the fish are — and don't forget rule number one (Daily Journal Q&A, 2017).
- IF a simple idea has been verified to work, THEN take it seriously and apply it with discipline for a long time, rather than seeking a cleverer one (Poor Charlie's Almanack, paraphrase-stable).
- IF an unpleasant necessary task competes with a pleasant one, THEN apply Granny's Rule: the unpleasant one first, daily (Talk Eleven, reward and punishment section).
- IF picking partners, employers, or heroes, THEN pick people who can be trusted without procedure, and become one — the seamless web of deserved trust (USC Gould, 2007).
